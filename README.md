# [Myth] Combat Simulator

This is the latest instalment of the Combat Simulator for Melvor Idle, the latest in a line of great but no longer maintained forks by [broderickhyman](https://github.com/broderickhyman/Melvor-Idle-Combat-Simulator-Reloaded), and [coolrox95](https://github.com/coolrox95/Melvor-Idle-Combat-Simulator), and [visua0](https://github.com/visua0/Melvor-Idle-Combat-Simulator-Reloaded)

This iteration of the combat simulation has been re-written from the ground up.
